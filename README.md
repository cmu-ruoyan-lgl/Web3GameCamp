# Web3GameCamp

Web3GameCamp 是一个专注于区块链游戏开发教育的在线学习平台。该平台旨在帮助开发者掌握 Web3 游戏开发所需的各种技能。

## 🚀 功能特点

- 💡 区块链基础教程
- 🎮 游戏开发实践
- 🎨 NFT 游戏资产创建
- 💰 GameFi 经济系统设计
- 👥 用户认证系统
- 📱 响应式设计

## 🛠️ 技术栈

### 前端
- React.js
- React Router
- CSS3
- JavaScript ES6+

### 后端
- Node.js
- Express.js
- MongoDB
- JWT 认证

## 🔧 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/web3gamecamp.git
Apply
Copy
```

2. 安装依赖
```bash
安装前端依赖
cd frontend
npm install

安装后端依赖
cd backend
npm install
```

3. 环境配置
```bash
在 backend 目录创建 .env 文件
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
PORT=5000
```

4. 启动项目
```bash
启动后端服务
cd backend
npm start
启动前端服务
cd frontend
npm start

```
